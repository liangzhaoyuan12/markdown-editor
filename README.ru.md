# Markdown Editor

Кроссплатформенный локальный Markdown-редактор с поддержкой macOS, Windows и Linux.

## Возможности

- 📝 Редактирование Markdown с живым предпросмотром и синхронизированной прокруткой
- 🔍 Встроенный поиск с подсветкой и быстрой навигацией
- ↩️ Отмена/Повтор (Ctrl+Z / Ctrl+Y)
- ⌨️ Горячие клавиши
  - `Ctrl+N` Новый, `Ctrl+O` Открыть, `Ctrl+S` Сохранить, `Ctrl+F` Поиск
  - `Ctrl+Enter` Вставить новую строку ниже
  - Панель инструментов: жирный, курсив, зачёркивание, цитата, заголовки, списки, ссылки, изображения, блоки кода, таблицы, эмодзи и др.
- 💾 Управление файлами (Новый, Открыть, Сохранить, Сохранить как) с автосохранением
- 🎨 Светлая/Тёмная тема, автоматическое отслеживание системной темы
- 🌍 Многоязычная поддержка: English, 简体中文, 繁體中文, 日本語, 한국어, Русский
- 🖥️ Кроссплатформенность (macOS, Windows, Linux)

## Технологии

- **Фронтенд**: Vue 3 + Vite
- **Десктоп**: Tauri 2.x
- **Рендеринг Markdown**: Rust (pulldown-cmark) через Tauri IPC
- **Бэкенд**: Rust

## Требования

- Node.js 16+
- Rust 1.70+
- Системные зависимости (см. [документацию Tauri](https://tauri.app/start/prerequisites/))

## Установка зависимостей

```bash
npm install
```

## Разработка

```bash
npm run tauri dev
```

## Сборка

```bash
npm run tauri build
```

После сборки исполняемый файл находится в директории `src-tauri/target/release/bundle/`.

## Использование

1. **Новый файл**: Нажмите кнопку «Новый» или Ctrl+N
2. **Открыть файл**: Нажмите «Открыть» или Ctrl+O, выберите локальный .md файл
3. **Редактирование**: Пишите Markdown в левой панели, предпросмотр справа в реальном времени
4. **Сохранить**: Нажмите «Сохранить» или Ctrl+S
5. **Сохранить как**: Нажмите «Сохранить как» для сохранения в новое место
